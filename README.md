# InnovaHire Marketing Site

InnovaHire LLC Marketing Website - Static site deployed to Cloudflare Pages.

## Cloudflare Pages Configuration

When setting up this project in Cloudflare Pages, use the following settings:

### Build Settings

- **Framework preset:** None (or Static Site)
- **Build command:** `npm run build`
- **Build output directory:** `dist`
- **Root directory:** (leave empty or set to project root)

### Environment Variables

No environment variables required for this static site.

## Local Development

```bash
# Install dependencies
npm install

# Run local development server
npm run dev

# Build for production
npm run build

# Deploy to Cloudflare Pages
npm run deploy
```

## Project Structure

- `public/` - Source files (HTML, CSS, images)
- `dist/` - Built output directory (generated by build script)
- `scripts/build.js` - Build script that copies public files to dist

## Build Process

The build script (`scripts/build.js`) recursively copies all files from the `public/` directory to the `dist/` directory, preparing them for deployment to Cloudflare Pages.
